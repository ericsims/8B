# 8 Bit TTL Computer
a simple 8 bit CISC computer built using TTL logic

## SW DEV

### python setup

`python -m venv .venv`
`source .venv/Scripts/activate`
`python -m pip install pip-tools`
`pip-compile.exe requirements.in`
`pip install -r requirements.txt`
